## 垂直对齐方式
#### 弹性盒子布局
```html
.item{
	display: flex;
	justify-content: center;
	align-items: center;
}
```


####
```html
.item{
	display: grid;
	place-items: center;
}
```


#### 
```html
.item{
	position: absolute;
	top: 50%;
	left: 50%;
	Transform: translate(-50%, -50%);
}
```
