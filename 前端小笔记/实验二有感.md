

# 选择器
### 元素选择器
选择使用行内元素style="color: aqua;
### id选择器
创建id标签之后在style中添加这种代码
#number{
		color: #00ffee;
		font-family: "微软雅黑";
		font-size: normal;
		align-content: center;
	}
### 类选择器
对于class元素，我们采用这种表达方式
.one{
    	color:rgb(0,255,238);
    	font-family: "微软雅黑";
    	font-size: large;
    	align-content: center;
    }
### 后代选择器
对于ul无序列表中的元素，如下：
<ul id="house">	

<li><a href="#">网站首页</a></li>
<li><a href="#">新房源</a></li>
<li><a href="#">二手房</a></li>
  <ul>
   <li><a href="#">出售房源</a></li>
   <li><a href="#">求购房源</a></li>
  </ul>
<li><a href="#">租房</a></li>
</ul>
我们可以采用CSS将其转化。
<style>
			ul li a{
				color: red;
				font-family: "宋体";
				background-color: yellow;
			}
</style>

### 子代选择器
对于div中的元素，我们可以采用div > p的方式访问
Term
:
 <div>
    	软件学院
        <p>软件工程系</p>
        <p>网络工程系</p>
        
   </div>
