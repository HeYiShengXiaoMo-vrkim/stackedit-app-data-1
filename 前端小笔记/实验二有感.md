

# 选择器
### 元素选择器
选择使用行内元素style="color: aqua;
### id选择器
创建id标签之后在style中添加这种代码
#number{
		color: #00ffee;
		font-family: "微软雅黑";
		font-size: normal;
		align-content: center;
	}
### 类选择器
对于class元素，我们采用这种表达方式
.one{
    	color:rgb(0,255,238);
    	font-family: "微软雅黑";
    	font-size: large;
    	align-content: center;
    }
