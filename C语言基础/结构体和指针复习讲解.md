

## 结构体
#### 结构体的声明
```c
struct student {
	int age;
	float score;
	char name[26];
};
```
像这样利用`struct`关键字加`student`结构体名称就构成了一个简简单单的结构体，这个结构体包含三个属性`age`,`score`,`name`，这里使用数组存放名称。
#### 结构体的内容
```c
struct student student_1 = {19,0,"liyingjie"};
```
这样我们就一次性定义了duo'ge
