请将一串长度最长为5的纯字母文本译成一个密码，密码规律如下：用原来的字母后面的第4个字母代替原来的字母。如C用G代替（如果该字母没后面没有第4个字母可代替，则需从26字母表中的首字母开始代替，比如：W用A代替，Z用D代替），最后得到的文本即为密码。

### 输入格式:

输入一串文本，长度最大为5。

### 输出格式:

输出对应的密码。格式为：  
password is 密码

### 输入样例:

```in
China
```

### 输出样例:

在这里给出相应的输出。例如：

```out
password is Glmre
```
