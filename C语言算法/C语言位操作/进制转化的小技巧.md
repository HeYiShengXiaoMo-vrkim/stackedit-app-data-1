## 如何写出别人看不懂的代码
### 乘法
 ```
 x *= 2//x <<= 1;
x >>= 2 // x /= 2;
x = (x << 1) + (x << 3) // x *= 10;
```
### 交换ab
```c++
a ^= b, b ^= a, a ^= b; //SWAP(a,b)
```
### 取出x二进制中的某一位
```c
for(int i = 7; i >= 0; --i)
cout << (x >> i & 1);//取到第七位
```
