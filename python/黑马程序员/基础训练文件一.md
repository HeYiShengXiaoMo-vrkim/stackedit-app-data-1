### 列表元素普遍加一百
```
list1 = [0, 1, 2, 3, 4, 5, 6, 7]
list2 = []
for i in list1:
	list2.append(i + 100)
print(list2)

优化后：
list1 = [0, 1, 2, 3, 4, 5, 6, 7]
list2 = [i+100 for i in list1]
print(list2)
```
