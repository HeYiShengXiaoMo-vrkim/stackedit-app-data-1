### 列表元素普遍加一百
```
list1 = [0, 1, 2, 3, 4, 5, 6, 7]
list2 = []
for i in list1:
	list2.append(i + 100)
print(list2)

优化后：
list1 = [0, 1, 2, 3, 4, 5, 6, 7]
list2 = [i+100 for i in list1]
print(list2)
```

### 两个列表合成一个新的
```
list1 = ["A", "B", "C"]
list2 = ["X", "Y", "Z"]

list3 = []
for i in list1:
	for j in list2:
		list3.append(i+j)
print(list3)

优化后：
newList = [i+j for i in list1 for j in list2]
print(newList)
```
