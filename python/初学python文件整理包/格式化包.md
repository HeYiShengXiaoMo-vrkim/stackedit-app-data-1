### 字符串格式化

```python
nums = [4, 5, 6]
msg = "Numbers: {0} {1} {2}".format(nums[0], nums[1], nums[2])
print(msg)  # 输出是 "Numbers: 4 5 6"

str = "{c}, {b}, {a}".format(a=5, b=9, c=7)  # 你可以通过这种方式给每个键赋值
print(str)  # 输出是 "7, 9, 5"
```

在这段代码中，我们展示了如何使用 `format` 方法进行字符串格式化。`format` 方法允许我们在字符串中插入变量的值。
